# cMedQA 数据集（用于健康问答联调）

来源：
- https://github.com/zhangsheng93/cMedQA

本目录文件：
- questions.csv.zip / answers.csv.zip：原始压缩包
- questions.csv / answers.csv：原始解压数据
- health_qa_sample_for_rag.md：抽样 200 条问答，适合直接上传到知识库做演示

建议测试步骤：
1. 在“知识库管理”里点击“新增文档”
2. 上传 health_qa_sample_for_rag.md
3. 点击“重建索引”
4. 在“健康问答”中提问如："失眠怎么调理"、"腰疼怎么办"，观察来源引用

备注：
- 该数据集为公开研究数据，请按其仓库说明使用。
